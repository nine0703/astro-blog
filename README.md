# 星澜科技企业门户

基于 Pinwheel Astro 主题整理的企业门户网站。当前版本保留原主题配色与布局风格，已收束为企业官网常用结构：首页、关于我们、解决方案、新闻动态、联系我们。

## 本地命令

```powershell
pnpm install
pnpm run dev
pnpm run build
```

## 内容入口

- 首页内容：`src/content/homepage/-index.md`
- 关于我们：`src/content/about/-index.md`
- 解决方案：`src/content/features/-index.md`
- 新闻动态：`src/content/blog`
- 联系我们：`src/content/contact/-index.md`
