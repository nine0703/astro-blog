# 澜心医疗科技企业门户

基于 Shadcn Studio Ink Astro 模板改造的中文医疗健康行业企业门户，定位为智慧医院、医疗数据治理与区域医疗协同方案展示站点。

## 当前内容

- 首页：中文企业门户首屏、行业洞察、咨询转化区。
- 联系页：中文表单与企业联系信息。
- 行业洞察：保留 4 篇医疗数字化主题文章，删除模板冗余内容。
- AI 客服：已接入本地 `embed.js`，并兼容 Astro 客户端路由切换。

## 常用命令

```bash
npm run dev
npm run build
npm run preview
```

本仓库不要求使用 pnpm；如需安装依赖，请按项目实际 Node/npm 环境处理。
